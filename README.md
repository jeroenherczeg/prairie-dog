# An Extraordinary Stream Generator for Laravel 5

## Installation

This package can be installed via Composer:

``` bash
composer require jeroenherczeg/prairie-dog
```

## Usage

```php
PrairieDog::forUser(1)->setPeriod('now','-1 hour')->generate();
```
